# 台灣旅遊達人 - 行程 Prompt 產生器 🎒

這是一個幫助你輕鬆規劃台灣國內旅遊行程的 Web 網頁應用程式。從「生活痛點」出發，解決每次安排旅遊都需要思考如何撰寫完整 Prompt 給 AI 的困擾。只需輸入基本的旅遊資訊與需求，即可一鍵生成結構完整、格式清晰的 Prompt，讓你能直接貼給 ChatGPT，獲得排版精美的行程建議！

此專案為 **2026_Vibe Coding 新手營：用 AI 開發辦公室系統** 的 Day 15 期末大魔王成果。

## 🌟 核心功能

*   **表單輸入**：提供直覺的介面輸入旅遊必備要素（出發/結束日期、旅遊地點、預算、實際感想/特定需求）。
*   **天氣小幫手（政府 Open Data API 整合）**：內建天氣查詢功能，介接政府開放資料平台，輸入縣市即可獲取天氣預報，輔助行程評估。
*   **自訂 AI 回覆格式**：可選擇 AI 生成的行程表格式（表格版、條列式版），方便直接貼上 HackMD 儲存。
*   **一鍵複製 Prompt**：自動將使用者輸入組裝成高質量的 Prompt 結構，點擊即可複製。
*   **PWA / 行動端支援**：支援行動端設備瀏覽，擁有專屬的 SVG 手機主畫面圖示（App Icon）。

## 🚀 成果展示與使用方式

1.  **開啟網頁**：使用瀏覽器打開 `index.html`。
2.  **查詢天氣**（選用）：在「天氣小幫手」區塊選擇要前往的縣市，點擊「查詢預報」來協助你決定行程安排。
3.  **填寫行程資訊**：依序填入出發日期、結束日期、旅遊地點及預估預算。
4.  **填寫實際感想與需求**：這部分很重要！輸入你的特殊需求，例如：「希望行程不要太緊湊，想多吃點在地小吃，會帶長輩同行...」。
5.  **選擇格式 & 產生**：選擇想要 AI 回覆的格式（表格或條列式），點擊「產生神級 Prompt」。
6.  **複製並前往 ChatGPT**：點擊「一鍵複製」，接著點擊下方連結前往 ChatGPT 貼上你的專屬 Prompt，讓 AI 為你規劃完美行程！

## 🛠️ 技術使用背景

*   **HTML5 & CSS3**：構築現代化毛玻璃（Glassmorphism）介面、RWD 響應式佈局。
*   **Vanilla JavaScript (ES6+)**：處理表單邏輯、字串模板組合（Prompt 生成）以及與外部 API 互動。
*   **Fetch API &非同步處理**：串接政府開放資料平台取得天氣資訊。
*   **Google Fonts & Boxicons**：使用 Noto Sans TC 提供美觀的中文字體，並以 Boxicons 點綴介面增加互動感。
